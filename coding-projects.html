Pizza App

Project description

This dynamic pizza ordering web app, built with React, provides a seamless user experience with the following features:

  • Search Functionality: Quickly search for pizzas by name in real-time.
  • Quantity Selection: Easily choose the desired quantity for each pizza.
  • Sold-Out Status: Automatically disables ordering for pizzas that are sold out.
  • Closing Time Logic: Prevents orders from being placed after the shop's closing time.
  • Clean Design: Features a modern and intuitive interface for hassle-free navigation.

Animelist Web Application

Project Description

The Animelist Web Application is a straightforward yet functional platform designed for anime enthusiasts who want a simple way to organize their viewing history. This web app enables users to create a personalized list of anime they have already watched, ensuring they can easily keep track of their viewing journey. Additionally, users can organize their anime collection by separating titles into its genre category. By allowing users to manually add titles and categorize them, the application provides a clear and effective method to manage and review the anime they have enjoyed over time. Its clean interface and focus on core functionality make it an ideal tool for users seeking a lightweight, no-frills solution for anime tracking.

Booklist Web Application

Project Description

The Booklist Web Application is a simple and practical tool designed for users to keep track of the books they own. This application allows users to create a personal catalog by adding books to their list. When adding a book, users are required to input key details such as the title, ISBN, and the number of copies they own, ensuring a well-organized and accurate collection. This streamlined system is perfect for book enthusiasts looking to maintain an inventory of their library.
