Augmented Reality Namecard Project

Project description

This project involves creating a two-sided namecard with front and back designs. Using computer vision integrated with Unity, the namecard serves as a marker. When scanned through a compatible application, it triggers the display of a 3D model directly on the namecard, regardless of which side is scanned. The project combines graphic design, 3D modeling, and AR technology to produce an interactive and visually engaging experience.
